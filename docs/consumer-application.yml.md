# Consumer Application YAML

이 문서는 [`consumer-application.yml`](/Users/ygpark2/pjt/ainsoft/rag/ainsoft-rag-spring-boot-starter/docs/consumer-application.yml) 의 설명용 복사본입니다.

- `graphExtraction.preset: technical`는 기술 문서용 추출 규칙과 기본 모델을 함께 적용합니다.
- `model`을 명시하면 preset 기본값보다 우선합니다.
- 소비자 앱에서는 자기 프로젝트의 `src/main/resources/application.yml`에 필요한 설정만 복사해서 쓰면 됩니다.
