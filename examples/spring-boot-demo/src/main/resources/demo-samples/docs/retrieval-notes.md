# 그래프검색 설계

그래프검색, 현장검색포털, 운영대시보드, 안전관제센터, 작업허가서
그래프검색은 현장검색포털, 운영대시보드, 안전관제센터, 작업허가서 사이의 관계를 하나의 서브그래프로 연결합니다.
운영자는 그래프 페이지에서 공통 엔티티와 문서 경로를 함께 탐색할 수 있습니다.

# 검색 전략

하이브리드검색, 벡터검색, 키워드검색, FalkorDB
하이브리드 검색은 키워드 검색, 벡터 검색, 그래프탐색을 조합해 근거 문서를 찾습니다.
