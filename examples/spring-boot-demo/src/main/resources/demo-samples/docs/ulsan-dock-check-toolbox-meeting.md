# 울산조선소 도크점검 TBM기록

안전관제센터, 현장검색포털, 운영대시보드, 울산조선소, 도크점검, 도크운영팀
TBM기록 41번 문서는 울산조선소 도크점검 작업을 위해 도크운영팀이 작성한 기준 문서입니다.
현장검색포털은 하이브리드 검색과 ACL 필터링으로 TBM기록, 작업허가서, 위험점검표를 함께 찾고 운영대시보드와 동기화합니다.
