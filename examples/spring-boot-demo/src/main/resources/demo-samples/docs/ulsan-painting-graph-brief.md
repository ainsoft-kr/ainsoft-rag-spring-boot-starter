# 울산조선소 도장작업 그래프브리프

안전관제센터, 현장검색포털, 운영대시보드, 울산조선소, 도장작업, 도장품질팀
그래프브리프 13번 문서는 울산조선소 도장작업 작업을 위해 도장품질팀이 작성한 기준 문서입니다.
현장검색포털은 하이브리드 검색과 ACL 필터링으로 그래프브리프, 작업허가서, 위험점검표를 함께 찾고 운영대시보드와 동기화합니다.
